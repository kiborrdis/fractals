# Step Number Rule

Creates custom animation sequences by defining discrete steps between a min/max range with configurable transition functions.

Each transition can use different easing functions (linear, sine, elastic, etc.) and durations, giving you precise control over how the parameter evolves over time.

Perfect for complex, choreographed animations where you need fine-grained control over timing and movement. Define exactly when and how each change occurs.
