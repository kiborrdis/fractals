# Distance-Based Changes

These parameters control how values vary based on distance from the viewport center, creating dynamic spatial effects.

**Available for:**
- **cxPerDistChange**: Complex constant real component
- **cyPerDistChange**: Complex constant imaginary component
- **rPerDistChange**: Escape radius variation
- **iterationsPerDistChange**: Iteration count variation
- **Mirroring distance changes**: Affect symmetry parameters

Creates gradients, zooming effects, and spatially-varying fractal characteristics across the image.
