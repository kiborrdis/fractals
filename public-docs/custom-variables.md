# Custom Variables

Define your own named variables to use in fractal formulas alongside built-in variables like `z` and `c`.

Each custom variable can have its own animation rule (static, range, step, or spline). Reference them by name in your formula - for example, `z^n + c` where `n` is a custom variable that you animate over time.

Perfect for parameterizing your formulas and exploring variations without rewriting the formula itself.
