# Radial Mirroring

Defines the angular spacing (in degrees) for radial symmetry reflections.

For example, 60° creates 6-fold symmetry, 45° creates 8-fold, and 90° creates 4-fold symmetry. Combines with other mirroring modes to create complex symmetric patterns.

Adjust with `radialMirroringPerDistChange` for angle variations across the viewport.
