# Color Gradient

Defines the color scheme used to visualize iteration counts across the fractal.

Colors map to how quickly each point escapes to infinity - creating the characteristic vibrant bands and smooth transitions. Add multiple color stops to create complex, beautiful gradients.

The gradient is the primary way fractals become visually stunning instead of just black and white.
