# Imaginary Value

The imaginary component (vertical axis) of a complex number in the complex plane.

In the complex constant `c = real + imaginary*i`, the imaginary value determines the up-down position. Adjusting the imaginary component shifts the fractal pattern vertically and affects which Julia set shape is generated.

Combine with the real component to explore different regions of fractal space.
