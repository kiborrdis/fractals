# Time Parameter

A dynamic variable that can be used in formulas to create animated fractals.

Reference `time` in your formula (e.g., `z^2 + c * sin(time)`) to make the fractal evolve over time. Perfect for creating mesmerizing animated sequences.

Set the initial time value or let it increment automatically for animations.
