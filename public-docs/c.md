# Complex Constant (c)

A fixed complex number parameter that determines the shape of Julia set fractals. Represented as a 2D vector `[real, imaginary]`.

In the fractal iteration formula `z_{n+1} = z_n^2 + c`, the `c` value stays constant while `z` varies per pixel. Different `c` values produce dramatically different fractal patterns.
