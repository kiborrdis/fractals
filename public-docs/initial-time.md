# Initial Time

Sets the starting value of the `time` variable when your animation begins.

By default, time starts at 0, but you can set any value. Useful for fast-forwarding to a specific point in a cyclic animation or creating variations of the same animation sequence.

Reference `time` in your formula (e.g., `z^2 + c * sin(time)`) to create animations that evolve over time.
