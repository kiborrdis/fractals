# Spline Rule (B-Spline)

Creates smooth, organic curves through control points using B-spline interpolation. Ideal for complex 2D parameter animations like complex constant paths.

Edit control points visually on a graph to design curved motion paths. The spline automatically creates smooth transitions through your points without sharp corners.

Perfect for creating flowing, natural-looking animations where parameters trace elegant paths through the complex plane or 2D parameter space.
