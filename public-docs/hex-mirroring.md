# Hexagonal Mirroring Factor

Controls the size of hexagonal cells when using hexagonal symmetry mirroring.

Larger values create bigger hexagonal tiles, while smaller values produce finer, more intricate hexagonal patterns. Works in conjunction with the hex mirroring mode to create 6-fold symmetry.

Adjust dynamically with `hexMirroringPerDistChange` for varying cell sizes across the viewport.
