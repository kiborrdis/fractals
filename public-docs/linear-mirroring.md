# Linear Mirroring Factor

Controls the spacing between mirror reflections when using square (linear) symmetry.

Determines how far apart the mirrored copies appear. Smaller values create tighter, more dense patterns, while larger values spread them out.

Use `linearMirroringPerDistChange` to vary the spacing dynamically across the image.
