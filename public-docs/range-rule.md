# Range Number Rule

Smoothly oscillates a numeric value between a minimum and maximum over a specified time period.

Creates cyclic animations where the parameter smoothly transitions back and forth. Set the cycle duration (seconds) and optional phase offset to control timing.

Ideal for breathing effects, pulsing parameters, or creating rhythmic variations in your fractal.
