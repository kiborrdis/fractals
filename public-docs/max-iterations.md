# Maximum Iterations

The maximum number of times the fractal formula is evaluated for each pixel before giving up.

Higher values reveal finer detail but increase rendering time. Lower values make rendering faster but may miss intricate structures. Typical range: 50-500.

Use `iterationsPerDistChange` to vary iteration count across the viewport, allowing higher detail in specific regions.
