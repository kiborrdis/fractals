# Band Smoothing

Controls the smoothness of color transitions between iteration bands in the fractal.

**Values:**
- **≥ 2**: Override auto-calculated smoothing power
- **< 0**: Disable smoothing (sharp bands)
- **0-2**: Use auto-calculated smoothing

Higher smoothing creates gradual, organic color flows. Disabled smoothing produces distinct, hard-edged bands.
